# GymLead Pro

Auto-generated by ETS-AI 10x.

## Idea Summary
A SaaS platform for small local gyms in India to capture, track, and close leads generated from WhatsApp and and Instagram.

## Problem Statement
Small local gyms in India struggle to efficiently capture, track, and convert leads originating from WhatsApp and Instagram.

## Target Persona
Indian small gym owner

## MVP Features
- WhatsApp Business API integration for lead capture and communication
- Instagram Direct Message integration for lead capture and communication
- Centralized lead inbox for all social media inquiries
- Lead profile creation and management (contact details, source, notes)
- Customizable lead statuses (e.g., New, Contacted, Follow-up, Converted, Lost)
- Basic CRM functionalities: lead assignment, activity logging (calls, messages)
- Quick reply templates for common inquiries
- Dashboard for an overview of active leads and their statuses
- User management for gym staff with role-based access

## Tech Stack
Python (Django/FastAPI) for backend, React/Vue.js for frontend, PostgreSQL for database, hosted on AWS/GCP/Azure, utilizing WhatsApp Business API and Instagram Graph API.
